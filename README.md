# FitLife Gym Business Website

## Project Overview

This project is a responsive business website built for a fictional fitness center called **FitLife Gym**.
The goal of the project was to practice modern frontend development concepts including semantic HTML, responsive layouts, and interactive JavaScript.

The website includes multiple pages and demonstrates how real business websites are structured.

---

# Pages

The website contains the following pages:

* Home
* About
* Services
* Contact

---

# Features

* Semantic HTML5 structure
* CSS Grid layout for page sections
* Flexbox for navigation and alignment
* Responsive design for mobile and desktop
* JavaScript menu toggle
* Contact form validation
* Accessibility using alt text for images

---

# Technologies Used

* HTML5
* CSS3
* JavaScript
* CSS Grid
* Flexbox

---

# What I Learned

### HTML5 Structure

I learned how to use semantic tags such as header, nav, main, section, and footer to create well-structured web pages.

### CSS Layouts

I practiced creating layouts using CSS Grid for page sections and Flexbox for navigation menus.

### JavaScript

I added interactive features like a mobile menu toggle and form validation.

### Accessibility

I ensured the website is accessible by adding alt text to images and using proper heading structure.

---

# Challenges & Solutions

### Challenge 1: Making the layout responsive

Solution: Used CSS Grid and media queries to adjust layouts for smaller screens.

### Challenge 2: Organizing project files

Solution: Created a proper folder structure separating HTML, CSS, JavaScript, and images.
